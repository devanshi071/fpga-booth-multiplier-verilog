
# FPGA Booth Multiplier (Verilog)

## Overview
This project implements a 4-bit signed Booth Multiplier using Verilog HDL with FSM-based control. The design is implemented on FPGA using Intel Quartus Prime.

## Features
- Booth’s Algorithm for efficient multiplication
- FSM (Finite State Machine) control
- Multi-cycle operation
- Signed number handling
- Output displayed on LEDs and 7-segment display

## Tools Used
- Verilog HDL
- Intel Quartus Prime
- FPGA Board

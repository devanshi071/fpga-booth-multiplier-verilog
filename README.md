# fpga-booth-multiplier-verilog
Implement a 4-bit signed Booth multiplier using the Verilog HDL with FSM control and FPGA implementation using Intel Quartus Prime, featuring real-time input and 7-segment display output.The multiplier will include a Finite State Machine (FSM) to manage the multiplcation process and the multiplier and multiplicand will be operated on over multiple clock cycles.Booth’s algorithm is used to decrease the number of additions and subtractions of the multiplier.
